# The MET Landing Page

Responsive landing page for The Metropolitan Museum of Art (The Met).
Built as a portfolio project using modern frontend tools.

## 🚀 Demo

👉 [Live Preview](https://aznayriy.github.io/landing_The_MET/)
🎨 [Figma Design](https://www.figma.com/design/lSR1m42L9YwzQwzzxKwHpw/THE-MET?node-id=8590-29&p=f&t=9D5WuIHsb9GM4aAa-0)

---

## 🛠️ Tech Stack

- HTML5
- SCSS (Sass)
- JavaScript (ES6)
- Parcel bundler
- GitHub Pages (deployment)

---

## 🎯 Features
 - Fully responsive layout
 - Clean and modern UI design
 - Interactive mobile menu
 - Smooth section structure
 - Optimized assets via Parcel

---

## 🚀 How to run locally

npm install
npm start

## 🏗 Build project

npm run build

## 🚀 Deploy

npm run deploy
